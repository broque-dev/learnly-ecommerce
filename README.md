# Learnly – Plataforma E-commerce de Cursos
Aplicación web para la compra y venta de cursos online.

## Tecnologías
- JavaScript
- Node.js
- MongoDB
- HTML, CSS

## Funcionalidades
- Registro e inicio de sesión
- Gestión de usuarios
- Sistema de mensajería
- Panel de administración
